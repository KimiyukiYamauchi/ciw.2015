# Lesson4 罫線とグラフィック要素

## 4.1 XHTMLで罫線を表示する

- 罫線(```<hr>```タグ) => 水平線のこと
- ```<hr></hr>``` = > ```<hr/>```
	- align => 位置(右寄せ(right)、左寄せ(left)、中央揃え(center))
	- size => 線の太さ
	- width => 線の長さ
	- noshade => 影付きの指定(非推奨)

## 4.2 画像とWebページ

- 画像(```<img>```タグ)
- ```<img></img>``` => ```<img />```
	- src => 画像ファイルのある場所
	- alt => 代替テキスト
	- align => 配置(画像とテキストとの配置、P148参照)
	- width => 画像ファイルの幅(横)
	- height => 画像ファイルの高さ(縦)
	- border => 枠の太さ

### 4.2.1 画像ファイルの形式

- JPEG
	- 非可逆圧縮、但し、JPEG2000 => 可逆圧縮
	- 写真や複雑な画像
- GIF
	- 256色、GIF87a、GIF89a
	- GIF89a
		- 透過性 => 画像の一部を透明にする
		- インターレース => 画像を少しずつ描画する
		- アニメーション
- PNG
	- 透過性、インターレース、可逆圧縮、アニメーション

### 4.2.2 画像へのalt属性の設定

- XHTML文書で画像を使用する時は、必ずalt属性とその値(代替テキスト)を  
コード中に記述しなければなりません．

### 4.2.3 背景画像と背景色の組み合わせ

- Webページ内には、背景画像と背景色をそれぞれ指定できます．何らかの理由で  
背景画像が表示されない時のためにも、両方とも指定しておくのが得策でしょう．

### 4.2.4 テキスト中に画像を配置する

- 画像をテキストとの位置関係で配置するためにalign属性を使用する

## 4.3 特殊文字

- ```&copy;``` => ©(著作権マーク)
- ```&reg;``` => ®(登録商標)
- ```&lt;```(lt => less than ～ => ～より小さい) => ```<```
- ```&gt;```(gt => greater than ～ => ～より大きい) => >
- ```&nbsp;``` => 半角スペース

### 4.3.1 半角スペース文字(ノーブレークスペース)

- 連続してスペースを挿入するのに使用する

## 4.4 色指定

<p><img src="fig6.png" width='50%' alt="Webページの構成"></p>
- 色の指定はRGBの順
- 個々の色は00(0)からFF(255)
- Webセーフカラーパレット => 216色の標準色
- ディザリング => サポートしていない色をRGB値の組み合わせで近似する機能

## 4.5 Webページの表示色と背景色

- ```<body>```タグの表示関係の属性
	- bgcolor => 背景色
	- text => テキストの色
	- alink => リンクをクリックした時の色
	- link => アクセスしていないリンクの色
	- vlink => アクセス済みのリンクの色
	- background => 画像の指定

## 4.6 フォント書式の指定

- ```<font>```タグ => 大きさ(size)、色(color)、書体(face)  
但し、非推奨 => CSSの使用が推奨

## 4.7 Webデザイン

### 4.7.1 配色

- Webサイトの雰囲気を決める重要な要素
- 色によってサイトの個性を伝えることができる

### 4.7.2 Webページのレイアウト

- ガイドライン
	- 簡潔にする
	- 各Webページヘのトピックを1つに絞る
	- ページを3つのセクションに分割する
	- 補助的なナビゲーションを提供する
	- コードの各セクションにコメントを入れておく

- 文書構造、```<div>```タグ、スタイルシート
	- ```<div>```タグのid属性 => スタイルシート(CSS)でレイアウト
	- テーブルの利用はNG

- 相対パス
	- XHTMLファイルが格納されているフォルダを基点として、下層にあるサブフォルダ  
	または上層のフォルダ(親フォルダ)を記述

- 空白、```<img>```タグ、XHTML
	- XHTMLではキー入力によるスペース挿入や改行を```<img>```タグ内や```<img>```タグ  
	の間で使うと、空白がページ内に表示されることがある => 不要な空白が追加されないように気をつける
